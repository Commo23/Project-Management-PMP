# PMP Flow Designer

Professional project management tool aligned with PMI/PMBOK standards. An all-in-one solution for managing projects using Waterfall, Agile, and Hybrid methodologies.

## Features

- **Multi-Project Management**: Create, save, and manage multiple projects
- **Project Flow**: Visualize project phases according to PMBOK 7th Edition
- **Kanban Board**: Advanced task management with drag & drop
- **Product Backlog**: User stories, acceptance criteria, and sprint planning
- **WBS (Work Breakdown Structure)**: Hierarchical project breakdown
- **RACI Matrix**: Roles and responsibilities management
- **Risk Register**: Comprehensive risk management
- **Stakeholder Matrix**: Stakeholder analysis and engagement
- **Team Management**: Dedicated team member management
- **Requirements Traceability Matrix**: Requirements management
- **Gantt Chart**: Timeline visualization
- **Charts & Reports**: Burndown, Velocity, Progress, and Risk trends
- **Dashboard**: Key metrics and KPIs
- **Collaboration**: Comments, mentions, project sharing
- **Integrations**: Slack, Jira, GitHub, and more
- **Internationalization**: English and French with official PMP terminology
- **Export/Import**: JSON, CSV, PDF formats

## Technologies

## Getting Started

### Prerequisites

- [Bun](https://bun.sh/docs/installation) installed

### Installation

```sh
# Step 1: Clone the repository
git clone <YOUR_GIT_URL>

# Step 2: Navigate to the project directory
cd Project-Management-PMP

# Step 3: Install dependencies
bun install

# Step 4: Start the development server
bun run dev
```

The application will be available at `http://localhost:8080`

### Build for Production

```sh
bun run build
```

## Project Structure

- `src/pages/` - Page components
- `src/types/` - TypeScript type definitions
- `src/utils/` - Utility functions
